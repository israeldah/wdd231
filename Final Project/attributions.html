Attributions

Ressources, API et outils utilisés dans la réalisation de SoundPulse.

API & Données

  • Deezer API — Données musicales (titres, artistes, albums, préécoutes audio). Source : developers.deezer.com. Utilisation gratuite pour les endpoints de recherche publics.
  • AllOrigins CORS Proxy — Proxy permettant les requêtes cross-origin vers l'API Deezer. Source : allorigins.win.

Technologies

  • HTML5, CSS3 (Custom Properties, Grid, Flexbox)
  • JavaScript ES2022 — Modules, Fetch API, async/await
  • localStorage Web API — Persistance des favoris et préférences de thème
  • Web Audio API — Lecture des préécoutes via l'élément <audio>

Polices & Icônes

  • Police système — Segoe UI, system-ui (aucune police externe requise)
  • Icônes — Émojis Unicode natifs (aucune dépendance externe)

Références & Apprentissage

  • MDN Web Docs — Documentation JavaScript, CSS, accessibilité ARIA
  • WAI-ARIA Authoring Practices — Bonnes pratiques d'accessibilité pour les modales et composants
  • Cours WDD 231 — BYU-Idaho Web Frontend Development I
